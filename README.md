# 粗利ランキング Dashboard

担当者別の粗利をリアルタイムで可視化するダッシュボードです。

## 🎯 機能

- **リアルタイムランキング**: 日次・月次の粗利ランキングを自動更新
- **トレンド分析**: 日別の推移をグラフで可視化
- **詳細ビュー**: 案件ごと・媒体ごとの粗利を一覧表示
- **自動更新**: 設定した間隔で自動的にデータを再取得
- **担当者管理**: 担当者の追加・削除が簡単

## 📦 セットアップ

### 1. GASプロキシの設定

1. [Google Apps Script](https://script.google.com/home/projects/create) で新規プロジェクトを作成
2. `gas-proxy.js` の内容をコピー＆ペースト
3. **デプロイ** → **新しいデプロイ** をクリック
4. 種類: **ウェブアプリ**
   - **実行ユーザー**: 自分
   - **アクセス**: 全員
5. **デプロイ** をクリックして承認
6. 表示された **ウェブアプリのURL** をコピー

### 2. ダッシュボードの起動

1. `index.html` をブラウザで開く
2. 初回起動時に設定画面が表示されます
3. コピーした **GAS URL** を貼り付け
4. **接続テスト & 開始** をクリック

### 3. 担当者の追加

1. 右上の **+ 追加** ボタンをクリック
2. 担当者名とスプレッドシートのURLを入力
3. **追加** をクリック

## 📊 スプレッドシートの形式

各担当者のスプレッドシートは以下の形式で作成してください：

### シート構成
- 1シート = 1案件
- シート名 = 案件名

### カラム構成
| A列（日付） | B列（媒体1） | C列（媒体2） | ... |
|------------|------------|-------------|-----|
| MM/DD      | TikTok     | Meta        | ... |
| 3/1        | 50000      | 30000       | ... |
| 3/2        | 45000      | 28000       | ... |
| ...        | ...        | ...         | ... |

### 対応媒体
- TikTok
- Meta
- Google
- LINE/LAP
- その他

## ⚙️ 設定

### 自動更新
- **ON/OFF**: トグルスイッチで切り替え
- **更新間隔**: 1, 3, 5, 10, 15, 30分から選択

### 担当者管理
- 設定画面から担当者の追加・削除が可能
- データは `localStorage` に保存されます

## 🔒 セキュリティ

- GASは自分のGoogleアカウントで実行されます
- GASアカウントにスプレッドシートの閲覧権限が必要です
- データはブラウザの `localStorage` にのみ保存されます

## 🛠️ トラブルシューティング

### データが取得できない場合
1. GAS URLが正しいか確認
2. GASアカウントにスプレッドシートの閲覧権限があるか確認
3. スプレッドシートの形式が正しいか確認

### エラーが表示される場合
- ブラウザのコンソールでエラー内容を確認
- GASのログ（実行ログ）を確認
